# Using ipconfig for Network Diagnostics

## Purpose

The `ipconfig` command is used in Windows Command Prompt to view and troubleshoot network configuration information.

---

## Commands Practiced

### Display IP Configuration

```bash id="17c8t8"
ipconfig
```

Used to check:

* IPv4 address
* Default gateway
* Subnet mask

---

### Display Full Network Details

```bash id="53e4s6"
ipconfig /all
```

Used to view:

* DNS servers
* DHCP status
* MAC address
* Adapter information

---

### Release Current IP Address

```bash id="q6s7i2"
ipconfig /release
```

---

### Renew IP Address

```bash id="n4f9l5"
ipconfig /renew
```

---

### Flush DNS Cache

```bash id="0f7z4u"
ipconfig /flushdns
```

Used during DNS troubleshooting scenarios.
