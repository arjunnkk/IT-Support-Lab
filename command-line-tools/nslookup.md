# Using nslookup for DNS Troubleshooting

## Purpose

The `nslookup` command is used to diagnose DNS resolution issues and retrieve domain name information.

---

## Commands Practiced

### Lookup Domain Information

```bash id="a9v2x7"
nslookup google.com
```

Used to retrieve DNS server response and IP address information.

---

### Test DNS Server Response

```bash id="w5j7m1"
nslookup github.com
```

Used to verify domain resolution functionality.

### Scenario — Website Not Loading

* Verified internet connectivity
* Used nslookup to test DNS resolution
* Confirmed whether domain name could resolve correctly
