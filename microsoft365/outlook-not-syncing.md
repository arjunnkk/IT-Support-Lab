# Outlook Not Syncing Emails

## Scenario

Microsoft Outlook was connected to the internet but new emails were not syncing properly.

---

## Symptoms Observed

* Inbox not updating
* Delayed email delivery
* Outlook showing sync issues

---

## Troubleshooting Steps

### 1. Checked Internet Connectivity

Verified internet access through browser testing and ping commands.

### 2. Restarted Outlook

Closed and reopened Outlook application.

### 3. Checked Account Status

Verified Microsoft account login and mailbox connection.

### 4. Reviewed Sync Settings

Checked Outlook send/receive settings and sync configuration.

### 5. Cleared Temporary Issues

Restarted the system and relaunched Outlook.

---

## Tools Used

* Microsoft Outlook
* Windows Settings
* Internet diagnostics
* Task Manager

---

## Outcome

Email synchronization restored successfully and mailbox updated normally.
