# PowerShell System Information Script

## Purpose

This exercise introduces basic PowerShell commands used in IT support environments.

---

## Commands Practiced

### Display Computer Information

```powershell id="2r5z8j"
Get-ComputerInfo
```

---

### View Running Processes

```powershell id="n7c1v4"
Get-Process
```

---

### Check Network Configuration

```powershell id="k3f8m0"
Get-NetIPAddress
```

* Network information gathering

## Outcome

Improved familiarity with PowerShell commands commonly used in IT support tasks.
