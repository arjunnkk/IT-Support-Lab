# Slow PC Troubleshooting

## Scenario

Windows 11 laptop experiencing slow startup and lag during normal usage.

---

## Symptoms Observed

* Slow startup time
* Applications taking longer to open
* High CPU usage during startup

---

## Troubleshooting Steps

### 1. Opened Task Manager

Checked CPU, RAM, and startup application usage.

### 2. Reviewed Startup Applications

Disabled unnecessary startup applications affecting boot performance.

### 3. Checked Storage Usage

Verified available disk space and storage health.

### 4. Restarted the System

Performed a full restart and monitored performance improvement.

---

## Tools Used

* Task Manager
* Startup Apps
* Windows Settings

---

## Outcome

System startup time improved and background resource usage decreased.

---
